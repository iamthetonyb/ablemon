## 1. Campaign Overview

Soccer fans aren't one buyer. This campaign deploys 12 ad variants, each built for a distinct psychological profile using NLP meta-programs and Spiral Dynamics targeting. Every variant in the SKILL.md toolkit is covered: all 3 copywriting frameworks, all 5 Spiral levels, both motivation directions, both frames of reference, all 4 sorting filters, and all 3 representational systems.

With $1,000, the goal isn't to run a full ~90-day campaign — it's to identify which NLP profile actually converts, which platform delivers the better CPA, and which audience cluster is worth scaling. This is a validation sprint. When the winner is clear, every additional dollar goes to the proven variant at proven targeting.

**What $1,000 buys:** a 21-day Meta sprint plus a 7-day concentrated TikTok test. Expected outcome: 1-2 winning Meta ad sets, 1 winning TikTok ad group, a CPA benchmark, and enough purchase or add-to-cart signal to scale the real winner without starving every test cell.

**Platform logic:**
- **Meta (Facebook + Instagram): $810** — primary purchase engine for this budget, with enough room to test 4 structured ad sets without starving the winner
- **TikTok: $190** — concentrated creative and conversion screen, not a 27-day always-on channel at this spend level

## 5. Campaign Setup Guide

### Meta Ads Manager — Step by Step

#### Step 1: Campaign Level

1. Go to Ads Manager → **Create**
2. **Objective:** Sales (not Awareness, not Traffic — keep the campaign objective as Sales and choose Add to Cart or Purchase at the ad set level based on signal quality)
3. **Campaign name convention:** `HF_WC2026_[Phase]`
   - Example: `HF_WC2026_P1_Discovery`
4. **A/B Test:** Leave OFF at this stage — spend control happens through ABO, not Experiments
5. **Advantage Campaign Budget (CBO):** Toggle **OFF** in Phase 1
6. **Campaign daily budget:** No campaign budget in Phase 1 — budgets are set at the ad set level
7. **Campaign spending limit:** Optional account-level guardrail only; do not use per-ad-set spend caps
8. **Bid strategy:** Lowest Cost (discovery phase)

#### Step 2: Ad Set Level

For each of the 4 ad sets:

1. **Ad set name convention:** `HF_WC2026_SetA_Planners` (or SetB/C/D)
2. **Conversion location:** Website
3. **Performance goal:** Maximize number of add-to-cart events
   - Only use "Maximize number of purchases" if the account already has stable recent Purchase volume
4. **Attribution setting:** 7-day click, 1-day view (Meta default — keep this)
5. **Budget:** Set at ad set level (ABO) — `Set A $12/day`, `Set B $12/day`, `Set C $12/day`, `Set D $12/day`
6. **Schedule:** Run for 5 full days in Phase 1, then consolidate
7. **Audience:** Apply targeting from Section 4
8. **Placements:** Manual placements (not Advantage+ Placements)
   - Set A, B, C: Facebook News Feed + Instagram Feed + Instagram Stories
   - Set D: Add Instagram Reels + Facebook Reels
9. **Audience controls:** Keep Wi-Fi-only OFF and do not daypart in Phase 1

#### Step 3: Ad Level

For each Phase 1 launch ad only:

1. **Ad name convention:** `HF_WC2026_V01_AIDA-OR-TW-INT` (tracking code as the ad name)
2. **Identity:** Connect to HoodieFlags Facebook Page + Instagram account
3. **Ad format:** Single image or video
4. **Creative:** Upload per Section 9 specifications
5. **Primary Text:** Copy from Section 2 (exact text)
6. **Headline:** Copy from Section 2
7. **Description:** Copy from Section 2
8. **Destination URL:** Use full UTM URL from Section 7
9. **Call to Action button:** Shop Now

Phase 1 live Meta ads:
- Set A: V01, V12
- Set B: V02, V10
- Set C: V03, V11
- Set D: V04, V05

Hold V06, V07, V08, and V09 in reserve for Phase 2 challengers only.

#### Step 4: Pixel & CAPI Configuration

This is non-negotiable before launch:

1. **Meta Pixel:**
   - Install via Meta's base code on all pages of HoodieFlags.com
   - Verify these 3 standard events fire: `ViewContent` (product pages), `AddToCart`, `Purchase`
   - Test with Meta Pixel Helper Chrome extension before going live
2. **Conversions API (CAPI):**

## 8. KPIs & Decision Framework

### Performance Benchmarks

| Metric | Kill Signal | Acceptable | Strong | Excellent |
|--------|------------|------------|--------|-----------|
| CTR (link) | Below 0.5% | 0.8–1.5% | 1.5–2.5% | 3%+ |
| CPC | Above $3.00 | $1.50–2.50 | $0.80–1.50 | Below $0.80 |
| CPM | Above $45 | $25–40 | $15–25 | Below $15 |
| Add-to-Cart rate | 0% after $40 | 1–3% | 3–5% | 8%+ |
| Purchase CVR | 0% after $60 | 1–2% | 2–3% | 4%+ |
| CPA | Above $45 | $30–45 | $20–30 | Below $20 |
| ROAS | Below 1.0x | 1.5–2x | 2–3x | 4x+ |

**Unit economics:** At $44.99/ea. product price with ~60-65% gross margin ($30-32 gross profit), you need CPA below $28-30 to be profitable. CPA above $40 means either the creative is wrong or the audience is wrong — go back to targeting before increasing spend.

### Decision Tree

```
After Phase 1 (Day 5):

Any ad set generating purchases?
  YES → Move to Phase 2. Scale what bought.
  NO  → Any add-to-carts?
          YES → Phase 2 at lower spend. Check product page.
          NO  → Pause all. Audit: is pixel firing correctly?
                Is landing page functional on mobile?
                Is the offer clear?
                Fix the bottleneck before spending more.

After Phase 2 (Day 15):

ROAS above 2x on winner?
  YES → Phase 3. Scale aggressively.
  ROAS 1.5–2x? → Phase 3. Scale conservatively.
  ROAS below 1.5x? → Do not scale. Test new creative variant before spending $350 more.

Which platform wins?
  Meta CPA lower → Shift Phase 3 fully to Meta. Kill TikTok.
  TikTok CPA lower → Unusual for e-commerce but possible with this demo. Reallocate.
  Equal → Keep both, weight toward whichever has better purchase CVR.
```

After Phase 2, filter GA4 by `utm_content` to see which tracking codes generated purchases. Map back to the coverage matrix in Section 2:

- If **Away-From variants** (V02, V05, V06, V09, V11) consistently outperform → the primary buyer is pain-motivated. Future creative should lead with the problem.
