Create paid advertising campaigns using the ATLAS Paid Ads skill.

Follow the paid-ads SKILL.md protocol:
1. Define campaign goals (awareness/traffic/leads/sales), target CPA/ROAS, budget
2. Structure campaign hierarchy (Campaign → Ad Set → Ad)
3. Apply naming convention: {Platform}_{Goal}_{Audience}_{Creative}_{Date}
4. **Deep NLP Profiling (CRITICAL)**: Define Tone_Social (Hook/Headline), Tone_Chronic (Pain/CTA), and Decision Strategy (Direct vs Derived) before writing.
5. Write ad copy using AIDA/PAS/BAB/4U frameworks matched to the NLP profile
6. Set up audience targeting and retargeting tiers
7. Budget: 70% proven / 20% new audiences / 10% new creatives

Platform limits:
- Google Search: 30 chars × 15 headlines, 90 chars × 4 descriptions
- Meta/IG: 40 char headline, 155 char description, 125 char primary
- LinkedIn: 70 char headline, 150 char primary

Creative testing: Message → Format → Hook → Visual (test in this order)
