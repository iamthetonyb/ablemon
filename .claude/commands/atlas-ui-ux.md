Generate a design system and build UI using the ATLAS UI/UX Pro Max skill.

1. Analyze requirements (platform, framework, light/dark, brand colors)
2. Generate design system tokens (REQUIRED — colors HSL, typography, spacing, shadows, radii, transitions)
3. Build components using design system (no ad-hoc styles)

- Performance: LCP < 2.5s, CLS < 0.1, lazy load below fold
- Responsive: mobile-first, test at 320px/768px/1024px/1440px
- Animation: 150-300ms, transform-only, respect prefers-reduced-motion
