# /atlas-copywriting $ARGUMENTS

Generate high-converting copy for: **$ARGUMENTS**

---

## Step 1: Extract inputs

From `$ARGUMENTS` or ask (ask only if truly ambiguous):

| Input | Required | Default if missing |
|-------|----------|-------------------|
| **audience** | Yes | Ask |
| **objective** | Yes (what action should they take?) | Ask |
| **medium** | Yes | email |
| **tone** | No | Match user's own message tone |
| **context** | No | None |

Do NOT ask for inputs that can be reasonably inferred from context.

---

## Step 2: Profile the audience

Before writing, mentally model the target:

**Meta-programs to detect:**
- **Motivation direction**: toward (wants gain) vs away-from (avoids pain/risk)
- **Frame of reference**: internal (trusts own judgment) vs external (wants social proof)
- **Chunk size**: big-picture (why/vision) vs detail-oriented (how/specifics)
- **Sorting filter**: self-interest (what's in it for me) vs other-interest (how does this help others)

Match the copy's language and framing to the dominant meta-programs.

---

## Step 3: Select framework

| Framework | Use when | Structure |
|-----------|----------|-----------|
| **AIDA** | Cold outreach, ads | Attention → Interest → Desire → Action |
| **PAS** | Pain-led, urgent | Problem → Agitation → Solution |
| **FAB** | Technical, benefit-led | Feature → Advantage → Benefit |
| **BAB** | Transformation story | Before → After → Bridge |

---

## Step 4: Select representational system

Match the dominant sensory language to the audience:
- **Visual**: "see the difference", "clear picture", "looks like"
- **Auditory**: "sounds right", "resonates", "in tune with"
- **Kinesthetic**: "feels right", "get a grip on", "solid foundation"

---

## Step 5: Write the copy

Apply all three layers simultaneously:
1. Framework structure
2. Meta-program language
3. Representational system vocabulary

**For emails specifically:**
- Subject line: <50 chars, specific, creates curiosity or urgency without clickbait
- Opening: pattern interrupt — don't start with "I" or a compliment
- Body: one idea per paragraph, no walls of text
- CTA: one clear action, no optionality ("reply yes" not "let me know if interested")

---

## Step 6: Scrub forbidden patterns

Remove ALL of these before delivering:

**AI fingerprints (never use):**
leverage, delve, robust, synergy, seamlessly, utilize, facilitate, paradigm, cutting-edge, game-changing, revolutionary, groundbreaking, innovative, holistic, streamline, elevate, empower, unlock, unleash, harness, supercharge

**Weak openers (never start with):**
"I hope this email finds you...", "I wanted to reach out...", "Just following up...", "I'm writing to...", "My name is..."

**Vague CTAs (never use):**
"Let me know if you're interested", "Feel free to reach out", "Don't hesitate to contact me"

---

## Step 7: Output

Deliver:
1. The copy itself (ready to send, no placeholders)
2. Framework used and why
3. Key meta-programs deployed
4. Alternative subject line (for emails)

If rewriting existing copy: show before/after side by side and explain what changed and why.

---

## Quality bar

Ask yourself before delivering:
- Would this get a response from a skeptical, busy person?
- Is every sentence earning its place?
- Does the CTA make the next step completely obvious?

If any answer is no, revise before delivering.
